# CSS3

##Selectors
- **Elements**  

    ```css
    htmlTag1, htmlTag2, htmlTag3,... {...}
    ```
- **IDs**  

  ```css
  htmlTag#tagID {...}
  ```
- **Classes**  

  ```css
  htmlTag.tagClassName {...}
  ```
- **Nesting**  

  ```css
  outerHtmlTag innerHtmlTag {...}
  ```
- **Children**  

  ```css
  parentHtmlTag > childrenHtmlTag {...}
  ```
- **Siblings**  

  ```css
  htmlTag + adjacentSiblingsHtmlTag {...}
  htmlTag ~ anySiblingsHtmlTag {...}
  ```
